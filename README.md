# 🛡️ CyberIdentitySentinel

CyberIdentitySentinel is an advanced, machine learning-powered identity verification platform. It analyzes user profiles, behavior, and network structures to detect impersonation risks and fake accounts in real-time.

## 🚀 Features
* **AI Copilot & Risk Engine:** Automated risk assessment using trained ML models (`fake_model.pkl`).
* **Impersonation Detector:** Matches behavioral analysis and username similarity metrics.
* **Interactive Frontend:** A beautiful dashboard built with Streamlit for tracking real-time profile metrics.
* **Comprehensive Test Suite:** Built-in unit tests for tracking graph structures, face similarity, and behavior verification.
